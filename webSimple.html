Lenguaje de Marcas

Intruduccion de lenguajes de marcas

Un lenguaje de marcas es un modo de codificar un documento donde, junto con el texto, se incorporan etiquetas, marcas o anotaciones con información adicional relativa a la estructura del texto o su formato de presentación. Es decir, es un tipo de lenguajes que nos permiten codificar o señalar qué tipo de información está contenida dentro de un documento.

HTML

El Lenguaje de Marcado de Hipertexto (HTML)El Lenguaje de Marcado de Hipertexto (HTML) es el código que se utiliza para estructurar y desplegar una página web y sus contenidos.

CSS

CSS significa Cascading Style SheetsCSS significa Cascading Style Sheets (Hojas de estilos en Cascada) y sirve para describir cómo los elementos HTML son mostrados en la pantalla o diferentes medios.
